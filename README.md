# Meteo-Film
Projet 3D avec intégration d'une vidéo et de météo grâce à des particules. L'utilisateur peut changer l'intensité de la neige et de la pluie. a partir d'une certaine quantité de neige, l'herbe disparait.

Projet réalisé en parfaite autonomie. 

Lien pour tester : https://play.unity.com/en/games/0ad00a6a-9108-4ecf-ba02-3baada9ef8a9/exportwebmeteo-film
